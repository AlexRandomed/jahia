To set-up Edit On pro (2.1 or 3.x) or Editize 2.0 Java HTML Editors for using with Jahia,
proceed to the following:

Edit-on Pro 2.1 (http://www.realobjects.de)
-------------------------------------------

1. Download the 2.1 release of Edit-on Pro here:
  
   	http://www.realobjects.de/downloads
  
   You will need to complete a form to request a download link and a trial
   license key.
  

2. Extract the content of the directory "eopro" from the original archive file
   to :
    	
    	TOMCAT_HOME/webapps/jahia/jsp/jahia/htmleditors/eopro


3. Put in the same directory the license file "licensekey.xml" you should have
   received from Real Object.
 
   
4. Open the following file :

	TOMCAT_HOME/webapps/jahia/WEB-INF/etc/htmleditors/htmleditors_config.xml
  
   and uncomment the following lines:

	<!--
	<editor>
		<id>eopro</id>
		<name>Edit On Pro 2.1 Applet</name>
		<base-directory>eopro</base-directory>
		<include-file>eopro_htmleditor.jsp</include-file>
		<client-groups>
			<client-group>IE</client-group>
			<client-group>MOZILLA</client-group>
		</client-groups>
	</editor>
	-->


5. Restart Jahia.

-------------------------------------------


1. Download Editize 2.0 here:

   	http://www.editize.com/download.php

   received from Editize.
