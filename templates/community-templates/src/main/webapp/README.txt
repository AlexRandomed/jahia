

Content of community templates directory :

Usage :
- .jsp file : standalone JSP file
- .jspf file : include file, relative to the context of the page

1 / folders :

- common : files shared with all jsp templates

- error : local error files (404, 403, 501) to fit the current site look and feel.

- images : contains preview.gif, to show a preview in template selector

- skins : directory contains avaible skins for skinnable content

- theme : directory contains avaible themes

2 / files :

- definitions.cnd : content definitions for current template set
- templates.xml : template list declaration for current template set
